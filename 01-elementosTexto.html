ELEMENTOS DE TEXTO

ENCABEZADOS

Los encabezados en HTML permiten poner títulos y subtítulos.. Los niveles van de <h1> a <h6>

<h1>Encabezado h1</h1>
<h2>Encabezado h2</h2>
<h3>Encabezado h3</h3>
<h4>Encabezado h4</h4>
<h5>Encabezado h5</h5>
<h6>Encabezado h6</h6>

PÁRRAFOS

Para los párrafos en HTML se utiliza <p>.
Si queremos usar un salto de línea usamos <br>
Si queremos añadir un espacio en blanco usamos  

<p>Esto es un párrafo</p>

Esto es un párrafo

TEXTO MUY IMPORTANTE

Para resaltar la importancia de un texto usamos <strong>

<p>Párrafo con <strong>texto muy importante</strong></p>

Párrafo con texto muy importante

TEXTO ENFATIZADO

Para enfatizar un texto usamos <em>

<p>Párrafo con <em>texto enfatizado</em></p>

Párrafo con texto enfatizado

TEXTO CON OTRO FORMATO

Para poder dar un estilo a una parte de un texto usamos <span>

<p>Párrafo con <span class="violeta">texto violeta</span></p>

Párrafo con texto violeta

ABREVIATURA

Para escribir una abreviatura <abbr>

<abbr title="Sistemas Microinformáticos y Redes">SMR</abbr>
SMR

CITA

Para escribir una cita <cite>

<p><cite>El Hobbit</cite>, de J.R.R. Tolkien</p>
El Hobbit, de J.R.R. Tolkien

DIRECCIÓN FÍSICA

Para escribir una dirección física <address>

<address> C/ Antonio Mª Concha<br> 10300 Navalmoral de la Mata<br> Cáceres </address>

CÓDIGO

Para poner código de ejemplo usamos <code>

<code> console.log ('Hola mundo'); </code>
console.log ('Hola mundo');

SUPER ÍNDICE Y SUB ÍNDICE

Para poner poner un super índice usamos <sup> y para un sub índice usamos <sub>

<p>100 m<sup>2</sup></p>
<p>H<sub>2</sub>O</p>

100 m2
