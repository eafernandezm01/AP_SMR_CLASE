LISTAS ORDENADAS

Para las listas ordenadas se utiliza <ol> y para añadir los items de la lista usamos <li>.

<ol>
<li>HTML</li>
<li>CSS</li>
<li>JS</li>
</ol>

  2. CSS
  3. JS

</ol>

<li>Item 3

</ol>
</li>

LISTAS DE DEFINICIÓN

Para las listas de definicion se utiliza <dl>. Para cada definición, usamos <dt> para añadir el término y <dd> para añadir su significado.

<dl>
<dt>Término 1</dt>
<dd>Definición 1</dd>
<dt>Término 2</dt>
<dd>Definición 2</dd>
<dt>Término 3</dt>
<dd>Definición 3</dd>
</dl>
