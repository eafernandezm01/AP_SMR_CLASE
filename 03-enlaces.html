ENLACES Y NAVEGACIÓN

ENLACES

La etiqueta <a>, permite crear un enlace a otra página, un correo o un teléfono.

<a href="http://www.google.com" target="_blank"> Google </a>
<a href="mailto:correo@correo.es"> mail </a>
<a href="tel:+34666555444"> 666555444 </a>
Ir a google
mail
666555444

El atributo target puede contener:

  • "_blank": La página se abre en otra pestaña.
  • "_self": En la misma pestaña (si no se especifica target, lo realiza sobre la misma pestaña)

NAVEGACIÓN

La etiqueta <nav>, permite crear un barra de navegación. Los enlaces a otras páginas se especifican dentro de la etiqueta.

<nav>
<a href="./01-elementosTexto.html">Elementos de texto</a>
<a href="./02-listas.html">Listas</a>
<a href="./03-enlaces.html">Enlaces</a>
</nav>
